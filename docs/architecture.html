Factory Inventory Management System

System Architecture — Tech Stack — Data Flow

Overview

A full-stack demo application for managing factory inventory, orders, procurement spending, demand forecasts, and backlog. The frontend is a Vue 3 SPA communicating via REST with a Python FastAPI backend. All data is served from in-memory JSON loaded at startup — no database is required.

Vue 3 + Composition API Python FastAPI Pydantic v2 In-memory JSON No database i18n: EN / JA
